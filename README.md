# Seven Lakes Tajikistan

Open source tourism website about the Seven Lakes (Haft Kul) located in the Fann Mountains of Tajikistan.

The website provides information about:

• Seven Lakes tour from Samarkand  
• Haft Kul tour  
• Fann Mountains jeep tours  
• Travel routes and lake descriptions
